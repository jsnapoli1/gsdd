# GSDD Execution Redirect

This public skill name is deprecated as a human entrypoint.

If a human invokes `gsdd-execution` directly, do not run the execution workflow here.
Tell them to use the `gsdd-execute` command instead.

Do not claim that `gsdd-execution` and `gsdd-execute` are different workflows.
Explain that `gsdd-execute` is the public command and the actual execution behavior now lives behind an internal skill used by that command.
